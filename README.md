# Projeto para a Disciplina de Gestão Organizacional

## Esse projeto tem como objetivo, desenvolver uma solução digital para o evento que será realizado no IFRN (Caso hipotético)

# Solução Digital

## Escolhemos desenvolver um site para programação do evento, todos os partipantes podem entrar no site e acompanhar as programações, horários, lugares das palestras e acompanhar o evento em tempo real através das lives que estão disponíveis no próprio site.

# Tecnologias

- Nest.js
- Next.js
- Prisma
